<h2>How to Import Workflow into n8n: </h2>

<ol>
  <li>Download the workflow JSON file from this repository</li>
  <li>Open your n8n dashboard</li>
  <li>Click <strong>"Import"</strong></li>
  <li>Select <strong>"From File"</strong></li>
  <li>Upload the downloaded JSON file</li>
  <li>Click <strong>"Import"</strong></li>
  <li>Update credentials (API keys, tokens, etc.)</li>
  <li>Activate the workflow</li>
</ol>
